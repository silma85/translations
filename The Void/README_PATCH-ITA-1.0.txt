ISTRUZIONI

* Funziona con tutti i tipi di distribuzione del gioco (incl. Steam) e 
  qualsiasi lingua originaria

* Copiare il file .zip nella directory \data del gioco (es. 
  C:\Programmi\Steam\steamapps\common\the void\data per un'installazione su
  Steam) ed estrarre nello stesso punto, tipicamente con clic destro -> "Estrai
  Qui". Sovrascrivere se richiesto
  
* Controllare che sia stata creata la gerarchia "\Properties\Strings" e che 
  nella cartella "Strings" siano presenti 15 file .txt e il file voiceover.dat

* Giocare alla versione con testi italiani del gioco.

============================================
NOTE

* Per limitazioni dei font di gioco, i caratteri accentati sono stati
  sostituiti dalle versioni "apostrofate": a', e', etc.

* E' meglio cominciare una nuova partita dopo aver applicato la patch; non
  farlo potrebbe portare a inconsistenze nei testi, soprattutto della
  guida in-game (non ho riscontrato problemi con i sottotitoli).

* IMPORTANTE: Questa patch si applica alla versione rilasciata come The Void (la versione internazionale).
  Non si applica a Tension o Turgor perchè i dialoghi sono diversi.
